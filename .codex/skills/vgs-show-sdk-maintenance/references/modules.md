# SDK Module Map

Core locations:
- Sources/VGSShowSDK/Core/APIClient: request pipeline and networking
- Sources/VGSShowSDK/Core/Show: VGSShow session and request flow
- Sources/VGSShowSDK/Core/Environment: environment and region config
- Sources/VGSShowSDK/Core/Analytics: analytics toggle client
- Sources/VGSShowSDK/Core/Session: session and headers

UI elements:
- Sources/VGSShowSDK/UIElements/VGSLabel
- Sources/VGSShowSDK/UIElements/VGSImageView
- Sources/VGSShowSDK/UIElements/VGSPDFView
- Sources/VGSShowSDK/UIElements/VGSViewRepresentableProtocols: SwiftUI representables

Other:
- Sources/VGSShowSDK/Utils: helpers and extensions
- Sources/VGSShowSDK/Erros: error types and mapping
- Sources/VGSShowSDK/VGSShowSDK.h: Objective-C surface
- Sources/VGSShowSDK/PrivacyInfo.xcprivacy: privacy manifest
